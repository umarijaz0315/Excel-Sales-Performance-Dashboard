# Excel Sales Performance Dashboard

## Project Overview
This Excel-based **Sales Performance Dashboard** was developed to transform raw retail sales data into actionable business insights. The dashboard provides a clear view of key performance indicators (KPIs), such as sales, profit, and quantity sold, and enables interactive exploration of trends, top-performing customers, and product categories.

### Key Features
- **Data Cleaning & Preparation:** Ensured accuracy and consistency of raw sales data  
- **KPI Analysis:** Tracked sales, profit, and quantity to monitor business performance  
- **Pivot Tables & Charts:** Performed detailed analysis using advanced Excel features  
- **Interactive Dashboards:** Created slicers and visualizations for dynamic data exploration  
- **Trend & Performance Insights:** Identified top customers, product categories, and sales trends  

### Skills Demonstrated
- **Excel (Advanced)**
- **Data Cleaning & Data Preparation**
- **Data Analysis & Reporting**
- **Data Visualization & Dashboarding**
- **Business Reporting & Decision Support**



### How to Use
1. Open the `Excel-Sales-Dashboard.xlsx` file in Microsoft Excel (2016 or later recommended).  
2. Use the slicers and filters to explore different time periods, product categories, and customer segments.  
3. Analyze KPIs and trends using the interactive charts and pivot tables.  

### Outcome
This project demonstrates the ability to **turn raw data into actionable insights**, improve decision-making, and showcase proficiency in **Excel-based data analysis and visualization**. Ideal for showcasing skills for an entry-level Data Analyst role.
